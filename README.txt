This program races shell and insertion sort on
randomly shuffled arrays, sorted arrays, and
reversed arrays to time which algorithm is better.


Running instructions

type make validate to determine if the sorting algorithms
work.

type make run to run the timing code



File contents

shellvinsert.py contains the timing code
myinsertion.py contains the code for insertion sort
myshell.py contains the code for shell sort
